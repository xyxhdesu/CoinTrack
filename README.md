# CoinTrack - 简洁高效的记账软件

CoinTrack 是一款专注于个人财务管理的记账软件，旨在帮助用户轻松记录收支、追踪财务状况，让理财变得简单高效。


## ✨ 功能特点

- **便捷记账**：快速记录收入与支出，支持多币种、多账户管理
- **分类管理**：自定义收支分类（如餐饮、交通、工资等），清晰掌握资金流向
- **数据统计**：通过图表直观展示收支趋势、分类占比，辅助财务决策
- **预算规划**：设置月度/季度预算，实时监控消费进度，避免超额支出
- **数据安全**：本地数据存储加密，支持备份与恢复，保障财务信息安全
- **轻量化设计**：简洁界面，低资源占用，流畅使用体验


## 🛠️ 技术栈

- 开发环境：Android Studio
- 编程语言：Java/Kotlin（基于 JDK 21）
- 架构：遵循 Android 最佳实践，可能包含 Jetpack 组件（如 Room、ViewModel 等）
- 数据存储：本地数据库（支持数据持久化）
